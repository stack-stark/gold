✦ 常见问题

你可能想了解的

无法安装插件怎么办？
+

请检查 VS Code 版本，确保 VS Code 或其衍生编辑器（Trae、Cursor 等）的版本不低于 1.96.0。

价格获取失败怎么办？
+

右键或使用 ALT+5 快捷键手动刷新价格。如果仍然失败，尝试重启编辑器。如果问题持续，可能是第三方接口暂时不可用，请等待恢复。

为什么没有收到提醒？
+

请检查是否开启了编辑器的"免打扰模式"。您可以关闭免打扰模式，或将插件加入提醒白名单。

支持哪些编辑器？
+

支持 VS Code 及其衍生编辑器，包括 Trae、Cursor 等，只要底层 VS Code 版本 ≥ 1.96.0 即可。

⚠️ 特别提醒：价格基于第三方接口获取，数据仅用于个人学习和研究目的。插件不做价格准确性、及时性保证，投资风险用户自负！
